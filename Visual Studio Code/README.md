# Visual Studio Code


## Install
- Windows `winget install --id=Microsoft.VisualStudioCode  -e`
- MacOS `brew install --cask visual-studio-code`
## Editor

- Split To  <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd><-</kbd> /<kbd>-></kbd>  
- Move To <kbd>Alt</kbd> + <kbd><-</kbd> / <kbd>-></kbd>
